# COLOR PALETTE GENERATOR

Simple RGB color generator for creating gradient color palettes

#### BUILT WITH

* HTML5
* CSS3
* React
* JavaScript
* Values.js library


#### DEMO

https://color-palette-generator-app.netlify.app/

#### FEATURES

* responsive
* copy color to clipboard
* color input
